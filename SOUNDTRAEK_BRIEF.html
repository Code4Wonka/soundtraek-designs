Color System: Monochromatic Depth

Each phenotype uses one color family, three tonal depths for luxury jacquard-like appearance:

  • Background: Primary hex at 100%
  • Prominent Motifs: Hex darkened 18% (~0.82 factor)
  • Subtle Motifs: Hex darkened 9% (~0.91 factor)

Icon Badge Prompts

Master Style Prompt

Prepend this block to EVERY icon prompt below:

Create a square icon for a music genre profile badge used in a running app. Style rules — follow exactly for every icon: • Format: 512×512px, transparent background • Shape: centered circular badge, slight drop shadow (soft, 20% opacity, 4px blur, 2px down) • Border: 3px solid ring in the icon's primary color, 8px inset from circle edge • Illustration style: bold flat vector illustration, clean shapes, no photorealism, no gradients on fills — solid colors only inside shapes • Line weight: consistent 2–3px outlines on all illustrated elements, same color as the darkest icon color • Composition: icon objects fill 55–65% of the circle interior, centered, slight upward bias • Colors: use only the specified palette (2–3 colors max per icon) — do not introduce additional colors • Background fill inside the badge circle: lightest color from palette at 12% opacity • Typography: none — no text, no numbers, no letters • Lighting: none — flat, no highlights or shadows on the objects themselves • Mood: confident, clean, modern — like a premium sports brand badge • Do NOT use: gradients, glow effects, lens flares, 3D effects, photographic texture, emojis as source material

Printful Specs & Production

Product Overview

Product Print Area Resolution Material Price Range
Shorts (Men's AOP) Full front + back 150dpi minimum 100% polyester mesh $35–45
T-Shirt (Unisex AOP) Front, back, sleeves 150dpi minimum 100% organic cotton $25–35
Windbreaker (AOP) All exterior panels 150dpi minimum Nylon/polyester blend $65–85

File Requirements

  • Format: PNG (transparent background, NO background color)
  • Color Space: sRGB (critical for accurate print colors)
  • Resolution: 150dpi minimum at final garment dimensions

Garment Dimensions

  • Shorts: 2550 × 1950 px (includes 0.5" bleed); badge icon 300×300px bottom right
  • T-Shirt: 3000 × 3300 px per panel (front/back/sleeves); optional 150×150px icons on sleeves
  • Windbreaker: Per-panel dimensions with 0.5" bleed; badge on upper back left or front left chest

Production Timeline

  • Week 1: Generate all icons (70 badges)
  • Week 2: Generate pattern tiles (~81)
  • Week 3: Color accuracy & file prep (tile patterns, place badges, export PNG)
  • Week 4: Order Printful samples, verify quality
  • Week 5: Batch upload to Printful, create Shopify store, launch
