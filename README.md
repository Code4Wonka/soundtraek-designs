A complete design collaboration brief for creating personalized music-culture apparel tied to user listening phenotypes.

## 📖 How to Use

**Generating Icons:**
1. Copy the Master Style Prompt from the Icon Prompts section
2. Append the individual phenotype icon prompt
3. Paste into Midjourney/DALL-E with suggested parameters
4. Export at 512×512px

**Generating Patterns:**
1. Select pattern prompts for your phenotypes
2. Use Midjourney (with `--tile` flag) or DALL-E (manual tiling)
3. Export seamless tiles ready for product dimensions
4. Desaturate/recolor to exact hex colors using Photoshop/Canva

**Preparing for Print:**
1. Tile seamless pattern SVGs to product dimensions (Photoshop/Canva/Figma)
2. Place badge icon in recommended location
3. Export as PNG, 150dpi, sRGB color space
4. Upload to Printful dashboard

## 🎨 Color Philosophy

Each phenotype uses **one color family with three tonal depths**:
- **Background** — Primary hex at 100%
- **Layer A** — Hex darkened 18% (×0.82)
- **Layer B** — Hex darkened 9% (×0.91)

This creates visual depth and wearability without multicolor complexity.
